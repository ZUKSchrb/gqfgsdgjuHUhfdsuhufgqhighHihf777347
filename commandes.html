Commandes du serveur Infinicube

Voici une liste des commandes que les joueurs peuvent utiliser sur le serveur Infinicube.


Commandes de connexion

    ! Si vous avez oublié votre mot de passe, contactez notre support !

    /login - Connecter à son compte Infinicube

    /register - Créer un compte Infinicube

    /changepassword - Modifier votre mot de passe de connexion actuelle


Commandes de base

  • /help - Affiche une liste des commandes disponibles. ( Bientôt retiré )
  • /msg - Envoie un message privé à un joueur.
  • /reply (ou /r) - Répond au dernier message privé reçu.
  • /spawn - Téléporte le joueur au point de spawn du serveur.
  • /home - Téléporte le joueur à son point de /home défini.
  • /sethome - Définit un point de retour avec la commande /home. ( MAX : 3 )
  • /delhome - Supprime un point de /home défini.
  • /tp - Demande la téléportation vers un autre joueur. ( Désactivé )
  • /tpa - Envoie une requête de téléportation à un autre joueur.
  • /tpaccept - Accepte une requête de téléportation reçue. ( Deviendra /tpyes dans le future )
  • /tpdeny - Refuse une requête de téléportation reçue. ( Deviendra /tpno dans le future )
  • /warp - Téléporte vers un point de téléportation prédéfini.
  • /back - Retourne à l'emplacement précédent. ( Bientôt retiré )
  • /mail - Envoie un e-mail au joueur est si le joueur n'est pas connécté il le recevra dés sa connexion

Commandes de groupe ou d'équipe

  • /team create - Crée une nouvelle équipe.
  • /team invite - Invite un joueur à rejoindre votre équipe.
  • /team join - Rejoint une équipe à laquelle on a été invité.
  • /team leave - Quitte l'équipe actuelle.
  • /team chat - Envoie un message privé à tous les membres de votre équipe.
  • /team home - Téléporte à la base de votre équipe.
  • /team sethome - Définit un point de /home pour toute l'équipe. ( Uniquement utilisable par le créateur de l'équipe )

Commandes d’économie

  • /balance - Affiche votre solde d'argent dans le serveur.
  • /pay - Envoie de l’argent à un autre joueur.
  • /shop - Ouvre le magasin du serveur.
  • /sell - Vend l'objet actuellement tenu dans votre main.
  • /auction start - Démarre une enchère pour vendre un objet. ( Bientôt utilisable )

Commandes de claim ou protection

  • ! Pour claim une zone utilisez une pelle en bois !
  • /unclaim - Libère la zone protégée actuelle.
  • /trust - Donne à un joueur la permission de construire dans votre zone protégée.
  • /untrust - Retire les permissions d’un joueur dans votre zone protégée.
  • /claimslist - Affiche une liste des zones protégées par vous.
  • /abandonclaim - Abandonne une zone spécifique.
  • /abandonallclaims - Abandonne tout vos claims ( Utilisable uniquement en dehors de votre claim ).

Commandes d'administration

  • /report - Signale un joueur pour mauvais comportement au staff. ( Bientôt retiré )
  • /rules - Affiche les règles du serveur. ( En finalisation )
  • /vote - Affiche les liens pour voter pour le serveur et recevoir des récompenses. ( Non disponible )

Commandes de météo et heure ( Inclus dans le grade Infini+ )

  • /day - Change l'heure pour le jour.
  • /night - Change l'heure pour la nuit.
  • /weather clear - Change la météo pour du beau temps.
  • /weather rain - Change la météo pour de la pluie.

Commandes diverses

  • /me - Permet de décrire une action dans le chat.
  • /list - Affiche la liste des joueurs connectés.
  • /afk - Informe les autres joueurs que vous êtes absent.
  • /ignore - Ignore les messages d’un joueur.
  • /ping - Affiche votre temps de latence avec le serveur.
